bio.dev — Multi-Tenant Portfolio Platform

FREE | Open Source | MIT License
By PinLab (spinarunsal)

bio.dev is a complete, production-ready portfolio builder built with Next.js 16, Prisma 6, NextAuth.js v5, and Tailwind CSS 4. Users register, choose a unique username, and instantly get a polished personal portfolio page — all managed from a sleek single-page dashboard with a live preview.

Why bio.dev?

  • No coding required — register and start building in seconds
  • Multi-tenant architecture — each user gets their own portfolio at /username
  • Modern glassmorphism design with smooth Framer Motion animations
  • Case study project pages — Problem, Solution, Results format
  • Built-in analytics — no third-party scripts needed
  • Cookie-free visitor tracking (GDPR-friendly)

Key Features

Dashboard

  • Split-panel layout: Editor (left) + Live Preview (right)
  • Profile editing with auto-save (800ms debounce)
  • Project CRUD with case study fields
  • Site settings, social links, and availability status
  • Visitor analytics with bar charts (7/14/30 days)
  • TR/EN bilingual content editing

Public Portfolio

  • Animated hero with typing effect
  • Categorized skill pills with hover animations
  • Markdown-rendered About section
  • Featured project cards with tech stack badges
  • Full case study pages (Problem, Solution, Results)
  • Contact and social link CTA buttons
  • Language toggle for visitors (TR/EN)
  • SSR for SEO optimization

Authentication and Security

  • NextAuth.js v5 with Credentials provider
  • JWT sessions with bcrypt password hashing
  • Middleware-protected dashboard routes
  • Reserved slug blocking (admin, dashboard, api, etc.)
  • Security headers (X-Frame-Options, CSP, Referrer-Policy)

Analytics

  • Cookie-free, fingerprint-based visitor tracking
  • Total views, unique visitors, daily breakdown
  • Top referrer sources and most visited pages
  • No third-party analytics dependency

Tech Stack

  • Next.js 16 — App Router, SSR, API Routes
  • TypeScript 5+ — Full type safety
  • Prisma 6 — Type-safe ORM with PostgreSQL/SQLite support
  • NextAuth.js v5 — Authentication and session management
  • Tailwind CSS 4 — Utility-first styling with custom glassmorphism
  • Framer Motion 12 — Smooth animations and transitions
  • Lucide React — Modern icon library

What's Included

  • Complete source code (Next.js 16 + TypeScript)
  • 7-model Prisma database schema (ready for PostgreSQL or SQLite)
  • Database seed script with sample data
  • Comprehensive HTML documentation
  • Environment configuration templates
  • Security headers pre-configured

Requirements

  • Node.js 18+ (20.x recommended)
  • npm 9+ or yarn
  • PostgreSQL 14+ (production) or SQLite (development)
